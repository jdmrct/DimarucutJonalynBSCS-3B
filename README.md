# DIMARUCUT-JF.-BSCS-3B

How to Run the Program

1. Go to this repository on GitHub
2. Click the green Code button → Download ZIP
3. Extract the downloaded ZIP file
4. Open the extracted folder and double-click DIMARUCUT-3B-CASE-STUDY.html
The simulator will open directly in your browser


Note: If the file does not open correctly, right-click it → Open with → select Google Chrome or any modern browser.
