# DIMARUCUT-JF.-BSCS-3B

How to Run the Program

Go to this repository on GitHub
Click the green Code button → Download ZIP
Extract the downloaded ZIP file
Open the extracted folder and double-click DIMARUCUT-3B-CASE-STUDY.html
The simulator will open directly in your browser


Note: If the file does not open correctly, right-click it → Open with → select Google Chrome or any modern browser.
